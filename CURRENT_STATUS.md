# Gebya Current Status

Gebya is a local-first PWA for Ethiopian small shops.

Known implemented or discussed areas:
- Today page
- Credit/Dubie customer ledger
- Reports
- More/Settings
- Dexie/IndexedDB offline data
- Quick Notes / shop memory
- Staff members and active staff attribution
- Staff sales totals
- Owner alert preferences
- Report search/filtering
- Backup/restore
- Early Phase 2 staff sale event sync planning/API persistence

Current field-test direction:
Use Gebya with real shop owners and staff to validate:
1. Whether quick sale capture is fast enough.
2. Whether staff can record sales from their own phone.
3. Whether owner wants immediate sale alerts or daily summary.
4. Whether reports help owner search old prices/items.
5. Whether Dubie and quick notes solve daily memory pain.

Current danger:
Overbuilding into POS, HR, payroll, inventory, or accounting before validating the shop memory assistant wedge.
