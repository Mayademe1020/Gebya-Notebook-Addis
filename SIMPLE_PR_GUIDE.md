## 🆘 If Something Goes Wrong

**"I can't find the Create Pull Request button"**
- Refresh the page
- Make sure you're on the compare page
- Look for a green button

**"The form shows many files"**
- That's normal - all your changes
- Just fill in title and description

**"There are conflicts"**
- GitHub will show a "Resolve conflicts" button
- Click it and use GitHub's UI to resolve
- Much easier than command line
